# Corporate Asset Inventory

A secure and efficient internal corporate system to track IT assets (laptops, monitors) and manage their assignments to employees. Currently under development as part of my Full-Stack portfolio.

### 🛠️ | Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (Fetch API)
* **Backend:** C# ASP.NET Core Web API, LINQ
* **Database:** MySQL (Entity Framework Core, Relational Data)

### 🚀 | Features (Planned)

* Manage corporate assets and employee records
* Track asset statuses (In Stock, Assigned, In Repair)
* Assign devices to specific employees (One-to-Many relationships)
* Dynamic search and filtering by name or serial number
* Input validation and basic security measures
